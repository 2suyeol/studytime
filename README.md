# 스터디로그 (StudyLog)

공부 시간을 측정하고, 공부한 내용을 노트로 정리하고, 그 노트로 **문제를 자동 생성해 스스로 풀어보는** 1인용 학습 웹앱입니다.

서버 없이 브라우저에서만 동작하며(`localStorage` 사용), GitHub Pages로 무료 배포할 수 있습니다.

## 기능

- **타이머** — 과목을 적고 스톱워치로 공부 시간을 측정 → 자동 저장. 오늘 총 공부량 / 과목별 통계 / 최근 기록 표시
  - **집중 모드**: 타이머가 도는 동안 다른 탭이 잠기고, 브라우저 탭을 바꾸거나 창을 최소화하면 자동으로 일시정지돼 자리 비운 시간은 공부 시간에서 빠집니다. 이탈 횟수도 함께 기록됩니다.
- **노트** — 공부한 내용을 작성·수정·삭제
- **문제 풀기** — 노트 내용으로 두 가지 문제를 자동 생성
  - **플래시카드**: 노트에 `질문 :: 답` 형식으로 적은 줄을 문제로 변환
  - **빈칸 채우기**: 일반 문장에서 핵심어를 가린 문제 생성

### 노트 작성 팁
```
이진 탐색의 시간복잡도는? :: O(log n)      ← 플래시카드 문제가 됨
스택은 후입선출(LIFO) 구조이다.             ← 빈칸 채우기 문제가 됨
```


## 로컬에서 실행

`index.html` 을 더블클릭해서 브라우저로 열면 끝입니다. 별도 설치가 필요 없습니다.

## GitHub Pages로 배포하기

1. GitHub에서 새 저장소(repository)를 만듭니다. (이름 예: `study-app`)
2. 이 폴더의 4개 파일을 저장소에 올립니다.
   - 웹에서 간단히: 저장소 페이지 → **Add file → Upload files** → 파일들 끌어다 놓고 **Commit**
   - 또는 git 명령으로:
     ```bash
     git init
     git add .
     git commit -m "first commit"
     git branch -M main
     git remote add origin https://github.com/<내아이디>/study-app.git
     git push -u origin main
     ```
3. 저장소 → **Settings** → 왼쪽 메뉴 **Pages**
4. **Source** 를 `Deploy from a branch` 로, 브랜치를 `main` / `/ (root)` 로 선택하고 **Save**
5. 1~2분 뒤 `https://<내아이디>.github.io/study-app/` 주소로 접속됩니다.

> 중요: 메인 파일 이름은 반드시 `index.html` 이어야 GitHub Pages가 자동으로 첫 화면으로 인식합니다.

## 데이터 보관 안내

모든 기록은 접속한 **브라우저에만** 저장됩니다(`localStorage`). 다른 기기나 브라우저에서는 보이지 않고, 브라우저 데이터를 지우면 함께 사라집니다.
